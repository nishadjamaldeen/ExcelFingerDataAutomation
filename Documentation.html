Automation Algorithm For Excel Finger Data

Internally Defined Functions

get_sd (path)

Params: path - takes in the path that will be used
Return: List of subdirectories in the given path


Operation: Looks into the path passed in as a parameter
and finds all available subdirectories



createExcelSheet(fileName, destinationPath)

Params: fileName - name of the file for the new sheet
destinationPath - location of the new file


Operation: Creates a fully formatted excel sheet for the
finger data



excelOperations(measurementFilePath, deformityFilePath, subfolder, compiledDataDestination, row)

Params: measurementFilePath - location of the file where finger measurements are stored
deformityFilePath - location of the table that contains the deformity data
subfolder - location of where the data is being stored
compiledDataDestination - location of final data file
row - nth row on which the new data will be added to the compiled file


Operation: Takes the data from each of the invidual excel sheets, transposes the data
and paste it to the nth row of the compiled data file



main()

Operation: Searches through the directory of files and loops through all the avaialble
excel files
